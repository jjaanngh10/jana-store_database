# jana-store_database
Project Overview
This project is a MySQL database designed for a simple store system
It manages products categories and basic store data using structured SQL 
The project demonstrates database design table relationships and SQL queries
Database Features
- Structured tables with primary keys
- Organized data for store management
- Ready to be connected with backend languages like Python
- Clean and reusable SQL script
